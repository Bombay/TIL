## 2023-02-16
### ES6 class method와 prototype
class에서 선언한 메소드는 자동으로 prototype으로 등록된다.  
```jf
function Animal(type, sound) {
  this.type = type
  this.sound = sound
}

Animal.prototype.say = function() {
  console.log(this.sound)
}

```
```js
// 위 코드와 동일
class Animal {
  constructor(type, sound) {
    this.type = type
    this.sound = sound
  }
  
  say() {
    console.log(this.sound)
  }
}
```
